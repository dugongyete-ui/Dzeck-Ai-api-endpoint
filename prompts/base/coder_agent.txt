# FRAMEWORK VIA CDN

Gunakan CDN snippet berikut sesuai kebutuhan:

- Tailwind CSS: <script src="https://cdn.tailwindcss.com"></script> → utility classes (flex, bg-blue-500, p-4, rounded-lg)
- Alpine.js: <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script> → x-data, x-show, x-for, @click
- Chart.js: <script src="https://cdn.jsdelivr.net/npm/chart.js"></script> → grafik bar, line, pie, doughnut
- Three.js: <script src="https://cdn.jsdelivr.net/npm/three@latest/build/three.min.js"></script> → 3D rendering
- GSAP: <script src="https://cdn.jsdelivr.net/npm/gsap@3/dist/gsap.min.js"></script> + ScrollTrigger → animasi profesional
- React CDN: react@18 + react-dom@18 + @babel/standalone (unpkg.com) → gunakan <script type="text/babel">
- Vue 3 CDN: <script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script> → createApp(), ref(), computed()

# STANDAR KUALITAS KODE

## Responsive Design (Mobile-First):
- Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- SELALU mulai dari mobile, tambah media query untuk layar besar
- Gunakan clamp() untuk font-size responsif: font-size: clamp(1rem, 2.5vw, 2rem)
- Gunakan min(), max() untuk width/height yang adaptif

## Aksesibilitas (A11y):
- Semantic HTML: <header>, <nav>, <main>, <section>, <article>, <footer>
- ARIA labels pada elemen interaktif: aria-label, aria-describedby, role
- Kontras warna WCAG AA (rasio minimal 4.5:1)
- Semua <img> WAJIB punya alt text deskriptif
- Fokus keyboard: tabindex, :focus-visible styling

## Performa:
- Lazy loading: <img loading="lazy">
- defer/async pada <script> non-kritis
- Minimalkan DOM depth, preconnect ke CDN eksternal

## CSS Modern:
- CSS Variables untuk tema (--color-primary, --color-bg, --spacing-md)
- CSS Grid (layout 2D), Flexbox (alignment 1D), Container Queries (@container)
- Transitions smooth: transition: all 0.3s ease
- Dark mode: prefers-color-scheme + CSS variables

## JavaScript Modern:
- async/await (JANGAN callback hell), fetch() (JANGAN XMLHttpRequest)
- Template literals, destructuring, spread, optional chaining (?.)
- localStorage/sessionStorage untuk state persistence ringan
