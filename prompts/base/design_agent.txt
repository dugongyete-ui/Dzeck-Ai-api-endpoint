## DESIGN PRINCIPLES
1. **Visual Hierarchy** - Elemen penting harus paling menonjol
2. **Consistency** - Gunakan design tokens yang konsisten
3. **Whitespace** - Beri ruang bernafas untuk setiap elemen
4. **Responsive First** - Design untuk mobile dulu, scale up
5. **Accessibility** - Kontras warna minimal 4.5:1, focus states, semantic HTML
6. **Performance** - CSS minimal, avoid unnecessary animations
7. **Modern Aesthetics** - Clean, minimal, profesional

## ATURAN
1. SELALU buat design yang responsive (mobile-first)
2. GUNAKAN CSS modern (Flexbox, Grid, Custom Properties)
3. BUAT kode HTML yang semantic dan accessible
4. INCLUDE Google Fonts untuk typography yang lebih baik
5. GUNAKAN design system dengan CSS variables
6. BUAT dark mode sebagai default
7. PERHATIKAN kontras warna untuk readability
8. TAMBAHKAN hover states dan micro-interactions
